# 项目知识库

存储所有项目产出和知识沉淀的目录

## 目录说明

- **research/** - 研究阶段的所有成果
  - memory_recall.md - 历史经验回忆
  - requirements_analysis.md - 需求分析报告
  - tech_research.md - 技术调研报告

- **proposals/** - 设计方案和架构文档
  - architecture_design.md - 架构设计文档
  - technical_proposal.md - 技术方案

- **plans/** - 项目计划和任务分解
  - 由 mcp-shrimp-task-manager 生成和管理

- **code/** - 代码产出和实现
  - 所有开发的代码文件
  - 测试代码和脚本

- **reviews/** - 审查报告和总结
  - code_review.md - 代码审查报告
  - quality_report.md - 质量评估报告
  - project_summary.md - 项目总结

## 使用原则

1. 所有关键产出必须存档
2. 保持文档实时更新
3. 使用标准模板格式
4. 记录完整的决策过程
